アクセサリファームウェア
===============

必要なもの
----
ArduinoベースのADK
SparkfunのMIDIシールド、もしくは互換のあるArduinoシールド

アクセサリを作る方法
----
* Arduino IDEでコンパイルします。 'Tool' → 'Board'メニューから、持っているADKに合ったボードの種類を選びます。 'Arduino MEGA 2560' もしくは 'Arduino MEGA(ATmega1280)' になるでしょう。
* MIDIシールドをADKボードから外します。そして、ADKをコンピュータにつなげます。
* ファームウェアをADKボードにアップロードします。
* ADKボードにMIDIシールドを付けます。MIDIシールドの'PROG/RUN'スイッチを'RUN'の側に設定します。
